Set your MultiMC path

To launch it through GOG Galaxy

Input the full path to MultiMC.exe or MultiMC.app.
(eg. "D:\Programs\MultiMC\MultiMC.exe" or "/Applications/MultiMC.app").

If you don't want to use MultiMC, leave the field blank.

If normal/vanilla Minecraft is not installed, you will not be able to install it via GOG Galaxy as Minecraft will show up as installed.

The path is invalid or leads to something that is not an executable, please try again.
