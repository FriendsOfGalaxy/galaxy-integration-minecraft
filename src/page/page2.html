Set your MultiMC path

To launch it through GOG Galaxy

The below is only if you want to use MultiMC. Otherwise, leave it blank and press 'continue'.

Input the full path to MultiMC.exe or MultiMC.app.
(eg. "D:\Programs\MultiMC\MultiMC.exe" or "/Applications/MultiMC.app").

If you use enable MultiMC support and normal/vanilla Minecraft is not installed, you will not be able to install it via GOG Galaxy as Minecraft will show up as installed.

The path is invalid or leads to something that is not an executable, please try again.
